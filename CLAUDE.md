# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 提供在此仓库中工作的指导。

## 仓库概述

这是一个中考作文范文与写作指导的合集，包含：

- **composition/**: 36 篇作文指导，包含考题分析、写作思路、写作大纲、范文和范文评析
- **押题/skills.md**: 用于生成作文写作指导的 AI 提示词模板

## 文件命名规范

作文文件遵循以下命名模式：`XX.标题.md`，其中 XX 是题号（01-36）。

题号对应 `composition/00.作文题目.txt` 中的作文题目。

## 文档结构

每篇作文文件（例如 `01.这是一次共赢.md`）包含以下标准化章节：

1. **作文题目**: 考试题目及要求
2. **考题分析**: 关键词、常见误区、得分要点
3. **写作思路**: 推荐角度和故事情节设计
4. **写作大纲**: 结构化章节分解及场景细节
5. **范文**: 600 字以上的范文示例
6. **范文评析**: 优点及写作技巧分析

## 关键约束

- 作文字数要求 600 字以上（中考作文要求）
- 需使用第一人称叙述视角
- 不得出现真实的学校名称或人名（考试规范）
- 优先选择记叙文体裁
- 诗歌体裁除外
