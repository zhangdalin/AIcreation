# 题目 1
阅读下面的文字，按要求作文

《现代汉语词典》对“共赢”的解释是:“大家都得到利益。”共赢，是合作中的彼此成就，是交往中的相互包容，是成长中的携手同行;它可以是一次小组合作的成功，一次朋友间的矛盾化解，一次与家人的相互理解，甚至是一次与自然的和谐相处.....

请以“这是一次共赢”为题，写一篇记叙文。

要求:(1)文中不要透露你个人的身份信息;(2)抄袭是不良行为，请不要照搬别人的文章;(3)不少于600字。

# 《这是一次共赢》作文全程指导

---

## 一、考题分析

**审题关键词拆解：**

| 关键词 | 理解要点 |
|--------|----------|
| **这** | 特指某一具体事件，须有真实感、现场感 |
| **一次** | 单一事件，叙事要聚焦，不可泛泛而谈 |
| **共赢** | 核心词——"双方"或"多方"都有所得，缺一不可 |
| **记叙文** | 文体明确，须有人物、事件、情节，有细节，有情感 |

**命题陷阱提示：**
- ❌ 只写"我"赢了，忽略"共"字
- ❌ 事件没有转折，平铺直叙
- ❌ 停留在表面行为，缺少内心情感刻画
- ❌ 结尾升华生硬，与正文脱节

---

## 二、写作思路

**立意方向选择（三选一）：**

> 🏆 **推荐立意**：与同学在学习合作中从"针锋相对"到"取长补短"，双方共同成长——**合作中的彼此成就**

其余可选方向：
- 与父母从对立到理解，亲子关系和解——家人间的相互理解
- 与曾经的对手化敌为友，共同赢得比赛——交往中的相互包容

**情节设计思路（以推荐立意为例）：**

```
矛盾引入 → 冲突激化 → 契机转折 → 携手合作 → 共同收获 → 感悟升华
```

**人物设定：**
- "我"：逻辑严谨，偏重理性分析，但不善表达
- "林晓"（对方）：文笔出色，思维发散，但做事粗心

两种风格碰撞，形成天然矛盾，又天然互补——为"共赢"埋下伏笔。

## 三、写作大纲

```
一、开篇入题（悬念式开头）
   ——以结果倒叙：展示那张贴在荣誉栏的奖状，引出回忆

二、矛盾起因（记叙文"起"）
   ——语文综合实践课，小组合作撰写调查报告
   ——"我"主张严格按框架写，数据优先
   ——林晓主张用散文化语言，情感优先
   ——两人意见相左，僵持不下，气氛冷凝

三、冲突激化（记叙文"承"）
   ——林晓独自修改，"我"愤而另起炉灶
   ——老师旁观不语，只说："好文章从来不是一个人写成的。"
   ——"我"心中有触动，却放不下面子

四、转折契机（记叙文"转"——最重要！）
   ——夜读偶遇"独学而无友，则孤陋而寡闻"
   ——想起林晓文章中某句打动自己的话
   ——主动找林晓和解，二人坦诚交流，发现彼此的不足与长处

五、携手合作（记叙文"合"）
   ——"我"负责数据逻辑与结构，林晓负责语言表达与润色
   ——细节描写：两人在灯下反复打磨的场景
   ——报告完成，老师点评："骨肉兼备，难得。"

六、结尾升华（点题）
   ——报告获奖，但"我"收获更多
   ——引用名句点题，揭示"共赢"的深刻内涵
```

## 四、范文

### 这是一次共赢

荣誉栏里那张红底金字的奖状，在走廊的阳光里静静发亮。每次经过，我总忍不住多看一眼——不是因为骄傲，而是因为那背后，有一段我几乎弄丢的故事。

那是初三上学期，语文综合实践课布置了一项任务：以小组为单位，完成一份关于"家乡传统习俗"的调查报告，纳入期末考核。我和林晓被分在同一组。

我与林晓，按理说是天造地设的搭档。我擅长收集数据、梳理逻辑，报告框架一向清晰；她文字灵动，笔下的句子有温度，读来如春风拂面。可偏偏就是这样两个人，在报告还没写到一半时，便起了龃龉。

"这里应该先列数据，用表格呈现，读者一目了然。"我指着草稿说。

"你这写的是实验报告还是调查报告？"林晓蹙眉，把稿子往旁边一推，"冷冰冰的数字，谁有耐心看？应该先写感受，先打动人，再说数据。"

"感性代替不了严谨。"

"严谨也掩盖不了无聊。"

谁也不让谁。最终，林晓独自修改了一份"她的版本"，我另起炉灶，写出了"我的版本"。两份草稿摆在桌上，老师看了许久，只淡淡说了一句话："好文章从来不是一个人写成的。"

我愣在那里，心里像压了一块石头，沉甸甸的，却说不清是什么滋味。

那晚，我翻开《礼记》复习，偶然读到一句话——**"独学而无友，则孤陋而寡闻。"** 我盯着这几个字，忽然有些坐立难安。我闭上眼睛，脑海中浮现出林晓稿子里的一段话，写的是村里老人在灶台边揉面，手上的茧子磨亮了岁月。那段文字，我当时嫌它"没有重点"，可此刻再想，却觉得有一种说不清的温柔，是我的数据表格里永远不会有的东西。

我拨通了林晓的电话。

沉默了几秒，她先开口："其实……你的逻辑真的比我强。"

我松了一口气，说："你的文字，我写不出来。"

就这样，我们在第二天放学后，并肩坐在教室最后一排，把两份草稿摊开，像两个工匠，耐心地把彼此最好的部分拼在一起。我搭骨架，她填血肉；我核实数据，她打磨语言。窗外的天色渐渐深了，教室的灯把两个人的影子拉得很长。

"这句话后面接数据，会不会太突兀？"她问。

"加一个过渡句，你来写。"我说。

"嗯。"

也不知道改了多少遍，当那份报告终于合上的时候，我们都没有说话，却都不约而同地笑了。那种感觉，像是两条原本平行的河，终于汇入了同一片海。

报告送交后，老师在课堂上点评，说了八个字：**"骨肉兼备，情理交融。"** 后来，这篇报告代表班级参加校级展评，拿了一等奖。那张奖状，贴在了走廊的荣誉栏里。

可我心里清楚，真正的奖赏，不是那张纸。

古人说：**"海纳百川，有容乃大。"** 我和林晓各自守着自己的"一条川"，便只能是两道细流；当我们愿意放下执念，彼此接纳，才汇成了可以奔涌的江河。赢，从来不只是一个人的事。真正的赢，是两个人都站在终点线上，都比出发时走得更远。

这一次合作，是我的赢，是林晓的赢，也是我们共同的赢。

而那份收获，至今仍压在我书桌最深处，比任何一张奖状，都要珍贵。

## 五、范文评析

### ✅ 亮点分析

**① 结构精巧，首尾呼应**
开篇以"奖状"引入，设置悬念，勾起读者兴趣；结尾再以"奖状"收束，但笔锋一转，强调精神收获重于物质奖励，主题得到深化，结构浑然一体。

**② 人物形象鲜明，矛盾设计合理**
"我"与林晓的性格差异（理性vs感性）符合生活逻辑，矛盾的产生自然，不刻意；两人的和解也有触发点（名句触动+彼此的暗自欣赏），不突兀，体现了人物的成长弧线。

**③ 细节描写到位，有生活温度**
> "窗外的天色渐渐深了，教室的灯把两个人的影子拉得很长。"

这一处环境描写，渲染了专注合作的氛围，也暗示时光流逝中的情感积淀，是记叙文"以景衬情"的典型运用。

**④ 古诗名句运用自然贴切**
- **"独学而无友，则孤陋而寡闻"**——出自《礼记》，在情节转折处引出，既合情境，又推动了"我"的心理转变，不显生硬。
- **"海纳百川，有容乃大"**——在结尾处以此升华主题，语言典雅，增强了文章的文化底蕴。

**⑤ 主题扣题精准，"共赢"二字落地有声**
文章没有停留在"我们赢了比赛"的表层，而是深入到"我们彼此成就了对方"的内核，真正写出了"共"字的分量，立意高于同龄人普遍水平。
