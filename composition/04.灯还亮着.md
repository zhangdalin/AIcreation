# 《平常中的美好》作文全程指导

---

## 一、考题分析

**材料解读：**

材料列举了六组生活中的平常细节：

| 场景 | 细节 | 蕴含的美好 |
|------|------|------------|
| 入学时 | 老师的问候 | 被看见、被接纳的温暖 |
| 百米冲刺 | 同学的呐喊 | 被支持、被陪伴的力量 |
| 餐桌上 | 饭菜喷香 | 家的味道、生活的踏实 |
| 台灯下 | 纸笔沙沙 | 专注的安静、成长的积累 |
| 枝头上 | 嫩绿新芽 | 生命的生机与希望 |
| 田野间 | 金黄麦浪 | 自然的馈赠、岁月的丰收 |

**材料核心句：**
> "这些平常真实的点点滴滴……往往蕴藏着生活的美好，让我们的心灵得到滋润，让我们获得前行的力量。"

**立意核心：** 美好不在远方，就藏在生活最平常的细节里；**用心发现**，平凡之处皆有光。

**审题注意：**
- ✅ 必须写**记叙文**，要有具体的人、事、场景
- ✅ 必须联系**初中阶段**的学习和生活经历
- ✅ 强调"**发现**"——不是美好主动降临，而是"我"有意识地去感受、去领悟
- ❌ 不能泛泛列举多件好事，要聚焦一两个场景写深写透
- ❌ 不能写成"感恩"主题的流水账，要有情感的起伏与顿悟的过程
- ❌ 标题不能直白如"生活是美好的"，要有文学质感

---

## 二、写作思路

**构思角度：**

材料给出的细节已经很丰富，写作时**不必面面俱到**，而要从中选取**一个最有感触的角度**，写出"我"从"视而不见"到"忽然发现"的心理变化过程——这个转变本身，就是文章最动人的地方。

**推荐选材：** 选取"台灯下，纸笔沙沙作响"这一意象为核心，写初三备考压力最大的一段时期，"我"在某个疲惫的夜晚，忽然被那一盏灯、那一声沙沙声、窗外的一弯月亮所触动，发现了平常生活里一直被忽略的美好与力量。

**情感主线：**
```
疲惫焦虑 → 麻木抗拒 → 偶然触动 → 用心感受 → 内心舒展 → 获得力量
```

**结构亮点：**
- 以"台灯"为贯穿全文的意象，首尾呼应
- 中间穿插母亲的一个细小举动作为触动点，以小见大
- 结尾从"一盏灯"延伸到"生活处处有光"，完成主题升华

---

## 三、写作大纲

```
一、开篇（以疲惫的夜晚切入，制造情感张力）
   ——初三深夜，台灯下，"我"盯着试卷发呆
   ——内心疲惫、焦虑，觉得生活只剩下刷题与考试

二、"视而不见"的日子（叙事"起"）
   ——描写那段时期的状态：行色匆匆，心不在焉
   ——饭不知味，景不入眼，对周围一切都失去了感知
   ——连母亲叫吃饭都觉得打扰，同学的玩笑也懒得回应

三、触动的瞬间（叙事"转"——核心段落）
   ——某夜，"我"又坐在台灯下，笔尖在纸上沙沙作响
   ——母亲轻轻推门，放下一杯热牛奶，什么也没说，又轻轻带上门
   ——那一声门轴转动的轻响，忽然击中了"我"
   ——抬起头，看见台灯的光晕、窗外的月亮、桌上那杯冒着热气的牛奶

四、重新"看见"的过程（叙事"合"）
   ——"我"放下笔，第一次在这段时间里，真正地看了看周围
   ——回想起这些天错过的细节：同学塞来的糖、老师课后轻拍肩膀、
     路边不知何时开了的花
   ——心里某种坚硬的东西，慢慢软化了

五、结尾（点题升华）
   ——重新拿起笔，却感到前所未有的平静与力量
   ——引名句点题：美好从未离开，是"我"的眼睛关上了太久
   ——"台灯还亮着"——结尾回扣意象，余韵悠长
```

## 四、范文

### 灯还亮着

那是初三最难熬的一段日子。

台灯开着，白光打在卷子上，一道解析几何题的辅助线我画了又擦，擦了又画，草稿纸上留下一道道灰色的痕迹，像这段时间我心里的样子——乱，看不清楚。

我不记得从什么时候开始，生活变成了这样一种状态：早上睁眼就想到今天有几张卷子，吃饭时脑子里还在默背单词，走在路上眼神空洞，连路边的树什么时候换了新叶都没注意到。母亲叫我吃饭，我应一声"来了"，却还是盯着书再看五分钟；同学在教室里讲笑话，我扯一下嘴角，算是回应。

那段日子，我像一台运转过热的机器，却忘了自己原本是有感觉的人。

那天晚上，我坐在台灯下，笔尖在草稿纸上沙沙地响。窗外很安静，偶尔有风吹过，树叶轻轻动一下，又归于平静。我揉了揉眼睛，盯着面前的题目，却什么都看不进去。

就在这时，门被轻轻推开了一条缝。

母亲没有说话，踮着脚走进来，把一杯热牛奶放在我的桌角，又踮着脚退出去，轻轻地把门带上。

那一声门轴转动的声音，轻得几乎没有。

可不知为什么，我的笔停了下来。

我就那么看着那杯牛奶，看着白色的热气一缕一缕地升起来，在台灯的光里飘散开。杯壁上印着一朵小花，那是我小时候最喜欢的那只杯子，母亲还记得。我忽然想到，这段时间，她每晚都会这样悄悄推门进来，放下一杯牛奶，然后悄悄出去——而我，从来没有抬过一次头。

我抬起头，第一次在许久之后，真正地看了看周围。

台灯的光晕是暖黄色的，把书桌染成了一片柔和的金色。窗玻璃上映着我的脸，窗外一弯月亮挂在深蓝色的天里，细细的，像一只眯起的眼睛。墙上贴着一张同桌写给我的便利贴，歪歪扭扭几个字："别怕，一起扛。"我记得她贴上去的时候我头也没抬，随口说了声"嗯"，便继续埋头刷题。

眼眶忽然有些发热。

我想起前几天语文课上读到的一句话——**"问渠那得清如许，为有源头活水来。"** 那时我只想着这是考点，现在却忽然觉得，那"源头活水"，也许就是这些平常的、细小的、几乎被我忘记的东西：一杯牛奶，一张便利贴，一弯月亮，一声沙沙的笔响。

古人说：**"草木有本心，何求美人折？"** 那些美好，从来不需要轰轰烈烈，它们只是静静地待在那里，等一个人肯低下头，肯慢下来，肯用眼睛好好地看一看。

是我的眼睛，关上太久了。

我端起那杯牛奶，喝了一口。是甜的，带着一点温热，从喉咙一路暖到了心里。我深吸了一口气，重新拿起笔，却发现那道困了我许久的解析几何题，思路忽然清晰起来了。

也许，人在太疲惫的时候，不是需要更用力，而是需要抬一次头，重新看见那些一直都在的光。

俗话说：**"灯不拨不亮，理不辩不明。"** 而我这才明白，有些灯，其实一直都亮着——只是我们低着头，走得太急，没有看见。

台灯还亮着，把我和这个安静的夜晚，都照得暖暖的。

## 五、范文评析

### ✅ 亮点分析

**① 标题意蕴丰富，一语双关**

"灯还亮着"——表层是台灯一直亮着，深层是"生活的美好从未消失，只是我们没有发现"。"还"字用得极妙，暗含转折与顿悟：不是美好消失了，是"我"的心暗了太久。标题克制而有余味，远胜直白的"发现生活的美好"。

**② 人物选材精准，以小见大**

全文没有选择宏大的感人事件，而是聚焦于**母亲放下一杯牛奶、轻轻带上门**这一极微小的动作。正是这种"轻"，才与"我"长久的麻木形成最强烈的对比，产生了击中人心的力量。这种以细节撬动情感的写法，是记叙文的高级技法。

**③ 心理描写层次分明，情感弧线完整**

全文情感经历了清晰的五个层次：
> 疲惫麻木 → 被触动 → 重新看见 → 内心松动 → 获得力量

每一层转变都有具体的场景支撑，不是空洞的"我突然想通了"，而是一步一步、有迹可循的心理流动。

**④ 古诗名句引用精准，与叙事融为一体**

- **"问渠那得清如许，为有源头活水来"**（朱熹）——在"我"重新感受生活美好的顿悟时刻引出，将"平常细节"比作"源头活水"，贴切而有深度，是全文立意的诗意表达。
- **"草木有本心，何求美人折"**（张九龄）——用来说明"美好不需要被寻找，它本来就在那里"，以物喻理，引用角度新颖，体现了较高的古诗鉴赏与运用能力。
- **"灯不拨不亮，理不辩不明"**（俗语）——在结尾处以俗语化用，语言亲切有力，与标题"灯还亮着"形成呼应，画龙点睛。

**⑤ 环境描写烘托情感，细腻而有层次**

> "台灯的光晕是暖黄色的，把书桌染成了一片柔和的金色。窗玻璃上映着我的脸，窗外一弯月亮挂在深蓝色的天里，细细的，像一只眯起的眼睛。"

这段环境描写，不是单纯的写景，而是"我"重新睁开眼睛之后的第一次真实凝视，景物的温柔与美丽，正是"我"心灵重新打开的外化，情景相融，浑然天成。

**⑥ 结尾回扣标题，余韵悠长**

末句"台灯还亮着，把我和这个安静的夜晚，都照得暖暖的"——以景收情，不直接说"我感受到了生活的美好"，而让那一片暖黄的灯光代替一切言语，含蓄而有力，给读者留下回味的空间。
