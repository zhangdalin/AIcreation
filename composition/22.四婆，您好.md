# 题目 22
读写共生(2025苏州中考)阅读下面的材料,按要求作文。(55分)

在小说《搬家》中，我们读到了小女孩枝儿的故事

假设现在枝儿已经长成和我们一般大小的中学生了，每次回忆往事，她都会想起那只大花鸡，想起当年的妈妈和四婆，觉得有很多话想跟她们说，她会说些什么呢?

请你以中学生枝儿的身份给妈妈或四婆写一封信，告诉她你最想说的话。

要求:(1)在“妈妈”“四婆”中任选一人作为你写信的对象，注意写信的口吻;(2)书信格式不作要求，不必拟写作文题目;(3)文中不要透露你个人的身份信息;(4)不少于600字。

## 一、考题分析

这是一道"读写共生"型作文题，考查阅读素材与写作表达的综合能力。难点有三：

1. **角色代入**：须以"中学生枝儿"的口吻写信，既有少女情怀，又有成长后的理性反思，切忌旁观者视角。
2. **书信体把握**：虽格式不作要求，但称呼、语气、情感流向须符合书信的私密性与真诚感。
3. **材料联结**：大花鸡、妈妈、四婆三个意象是全信的情感锚点，须自然融入。

**选择对象：写给四婆。** 人物距离感更大，反而给情感留白留出空间；老人形象与大花鸡更易产生温暖厚重的乡土气息，更难落俗套。

---

## 二、写作思路

以"大花鸡"为全信情感线索，层层推进：
- 开头：大花鸡的叫声唤起记忆，引出对四婆的思念；
- 中间：①搬家前与四婆共处的温暖日常；②搬家那天克制的离别场景；③成长后的感悟升华；
- 结尾：回扣开头细节，首尾呼应，余韵悠长。

---

## 三、写作大纲

```
一、开头
   - 大花鸡的叫声打开记忆之门

二、主体第一层：搬家前的温暖岁月
   - 四婆喂鸡的场景（细节描写）
   - 四婆说的一句话，温暖定格

三、主体第二层：搬家那一天
   - 离别场景：克制的悲伤，不煽情
   - 四婆与大花鸡的定格画面

四、主体第三层：成长后的感悟
   - 四婆留下的精神财富（勤劳、坚韧、温柔）
   - 大花鸡意象升华

五、结尾
   - 思念与祝愿，回扣开头
```

## 四、范文

四婆，您好。

每次提笔想给您写信，脑海里第一个跳出来的，不是您的脸，而是那只大花鸡的叫声——"咯咯咯"，浑厚而清脆，像一枚石子投进了我记忆的深水里，涟漪一圈圈散开，散不尽。

那是我们还住在老屋的时候。

您每天天不亮就起身，手里提着竹篮，里头装着米糠，走到院子里，嘴里念着谁也听不懂的词儿，大花鸡就从鸡圈里一拥而出，咕咕咕地绕着您转。我趴在窗台上看，总觉得那只鸡比我还懂事——它知道找谁，知道跟谁亲。俗话说"家有一老，如有一宝"，我那时不懂这话，只觉得您就是老屋的魂，有您在，连那只鸡都活得有滋有味。

您喂完鸡，回身看见我，就笑，笑纹从眼角铺开，像老屋院子里那棵皂角树的纹路，深而温暖。您说："枝儿，起来了？锅里有粥，热的。"

就这一句，热的。

四婆，我现在才明白，您说的那个"热的"，不只是粥的温度。

我不记得搬家是哪一天，只记得那天的天色灰蒙蒙的，像宣纸洇了水墨，说不清的惆怅。妈妈在屋里收拾箱子，我站在院子里，看着大花鸡还在悠闲地啄着地，像什么都不知道。可我知道，它留下来了，留在老屋，留在您这里。

您站在门槛上送我们，也没说什么，就那么看着。我回头望您，望见您的手轻轻搭在大花鸡的背上，鸡扑了扑翅膀，没跑，安静地蹲着。那个画面，我一直带着它，走过了小学、走过了初中，走到今天。

"露从今夜白，月是故乡明。"我在语文课上读到杜甫这句诗时，心里猛地一颤。故乡是什么？对我来说，故乡就是老屋的院子，就是那只大花鸡，就是您弯腰撒米糠时的背影。

我长大了一些，开始懂得您教给我的道理。您从不讲大道理，却用每一个清晨的行动告诉我：平凡的日子，也要活得认真。那只大花鸡吃糠咽糠，却每天都叫得那么响亮，生命力旺盛得让人心疼。我想，我也要做那样的人——不问出处，不惧平凡，活得热烈。

"天行健，君子以自强不息。"这话我也读到过。可我觉得，您不懂这些字，却早已活成了这个样子。您眉眼间的那份平静，不是麻木，而是历尽千帆后的从容，那是真正的自强。

四婆，我有时候想，要是大花鸡还在，要是老屋还在，要是我还能趴在那个窗台上，看您提着竹篮走进晨光里——那该多好。

但我知道，有些东西搬走了就回不来。不过没关系，因为您把最重要的东西，都已经装进了我心里。

无论走到哪里，我都不会忘记——锅里有粥，热的。

&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;想念您的枝儿
&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;二○二五年春

## 五、范文评析

**总体评价**：立意深远，情感真挚，语言精炼有诗意，约800字，整体达中考**50～55分满分层次**。

**亮点解析**：

| 技法 | 体现 |
|------|------|
| 细节以小见大 | "手里提着竹篮……嘴里念着谁也听不懂的词儿"，生活化细节胜过直白叙述 |
| 意象三次递进 | 大花鸡：童年玩伴→离别见证→精神象征，层层升华 |
| 古句嵌入自然 | 先抒情再引诗，水到渠成，不显堆砌 |
| 回环结构 | "锅里有粥，热的"首尾呼应，余韵悠长 |
| 口吻准确 | 少女天真与成长感悟并存，无说教感 |

**核心写作技法**：
> 书信体作文秘诀：**一个物象贯穿始终 + 两三个细节定格画面 + 古句点睛不堆砌 + 结尾回扣开头**。情感不需要多，真诚就够。
