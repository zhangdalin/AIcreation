# 题目 24
(2025陕西中考)阅读下面的材料,按要求作文。(55分)

外出，打声招呼;归来，报个平安。

田垄间，与父母一起劳作挥汗;厨房里，为家人备好一顿晚餐。

遇亲以礼，待友以诚;温良谦恭，和睦邻里。

Welcome to China.CanI help you?为外国游客送上热情的问候。

走出国门，自信、包容、理性的你，就是一张行走的中国名片。

----15岁，这样的你，其实很美！

请从你校园之外的生活中，收集自己“美”的证据，写一篇文章。它将是你初中生活最漂亮的书签，也将成为你高中生活崭新的扉页。

要求:(1)文中不要透露你个人的身份信息;(2)抄袭是不良行为，请不要照搬别人的文章:(3)不少于600字。

## 一、考题分析

本题为**材料引发型自命题作文**，文体不限（诗歌除外）。材料列举了五个场景，共同指向一个核心：**15岁少年在日常生活中呈现出的内在之美**。

**审题要点：**

1. **"美"的定性**：题目中的"美"不是外貌美，而是**品格美、行为美、精神美**——礼义、孝悌、担当、包容，这是立意的核心，切忌写成颜值、才艺的"美"。
2. **"证据"二字是关键**：要求写出具体的事件、细节、行为，不能泛泛抒情，须像"出示证据"一样，用事实说话。
3. **"校园之外"是限制**：明确规定取材于校园以外的生活，写家庭、社区、旅途、公共场所均可，切忌写课堂、考试等校内场景。
4. **材料的层次**：从家庭礼仪→劳动担当→待人接物→跨文化交流→国家名片，层层递进，暗示文章可以从小到大、由私域到公域展开。

**写作策略**：以记叙文为主，选取**两到三个不同维度的"美的证据"**，既有家庭温情，又有社会担当，用一个中心意象或情感线索将其串联，避免写成流水账。

---

## 二、写作思路

以**"书页"**为贯穿意象（呼应题目"书签""扉页"），将三个生活场景比作书中值得珍藏的三页，层层展开：

- **开头**：以"书"的意象切入，引出收集"美的证据"的主题；
- **主体第一层**：家庭之美——与父母劳作，感受孝与担当；
- **主体第二层**：人际之美——对陌生人施以善意，感受温良与诚；
- **主体第三层**：家国之美——为外国游客指路，感受文化自信与包容；
- **结尾**：三页合成一本书，呼应题目，升华至15岁少年独有的美。

**引用规划：**
- "百善孝为先"——写家庭劳作中的孝道；
- 孟子"老吾老以及人之老，幼吾幼以及人之幼"——写对陌生老人的善意；
- 《礼记》"礼者，天下之纲纪也"——写礼义之美；
- 林语堂"文化像一张名片，礼貌是正面，善良是背面"（化用）——写对外形象；
- 杜甫"烽火连三月，家书抵万金"化用——写归来报平安的感动。

## 三、写作大纲

```
一、开头
   - 以"翻书"的动作引入，引出"美的证据"的命题
   - 点明：美，不在镜中，在生活的褶皱里

二、主体第一篇"证据"：家庭劳作中的美
   - 场景：暑假跟父母下地，挥汗如雨
   - 细节：父亲的眼神、手上的茧、自己弯腰的动作
   - 升华：孝与劳动，是最素朴的美

三、主体第二篇"证据"：善待陌生人的美
   - 场景：街头扶起摔倒的老人/帮老奶奶提重物
   - 细节：对方的谢声，自己心里涌起的暖
   - 引用：孟子名言

四、主体第三篇"证据"：文化担当的美
   - 场景：旅游景区遇到外国游客，主动上前用英语引路
   - 细节：对方惊喜的眼神，自己说出第一句话时的忐忑与自豪
   - 升华：我们就是中国的面孔

五、结尾
   - 三页"证据"合成属于15岁的书
   - 呼应题目"书签""扉页"，收束有力
```

## 四、范文

**收集美的证据**

有人说，美需要寻找。我却觉得，美需要收集——像拾贝壳的孩子，蹲下身，在生活的沙滩上，一枚一枚，认真地拾。

这个暑假，我拾到了三枚。

**第一枚，在田垄间。**

外婆家在乡下，每逢暑假，父母都会带我回去帮忙。那天早上五点多，天色还带着一丝靛蓝，我们已经站在了玉米地里。父亲弯腰锄草，动作利索，汗水顺着脸颊滴进泥土，他也不擦，只是抬头对我笑笑："来，跟着爸学。"

我握着锄头，第一锄就歪了，差点锄到自己的脚。父亲走过来，手把手地教，他的手掌宽厚而粗糙，像树皮，也像盾。

"百善孝为先"，我一直以为孝是逢年过节的礼品、是饭桌上的嘘寒问暖。那一天，我才明白，孝是弯下腰、出一身汗，是和父母肩并肩，是不让他们孤独地站在田垄上。

那个早晨，我流着汗，心里却是甜的。那是我收集到的第一枚美的证据。

**第二枚，在街角处。**

那是一个下午，我独自去买书。路过一个十字路口，看见一位白发老奶奶提着两大袋东西，步履蹒跚，走几步歇一下，额头上满是汗。

我停了一下——只是一下。然后走上去："奶奶，我帮您提吧，您去哪儿？"

老人愣了愣，随即松开了手，"谢谢你，孩子，就在前面那个小区。"

一路上，老人说她腿不好，儿子在外地，我只是默默地提着袋子，听她说话，偶尔应两声。到了小区门口，她转身看着我，说："你这孩子，真好。"

孟子说："老吾老以及人之老。"那一刻，我忽然觉得，善良不需要理由，善良本身就是答案。老人离去的背影，成了我那天傍晚见过的最美的风景。

这是我收集到的第二枚证据。

**第三枚，在古城墙下。**

暑假末，我们去西安旅游。城墙下，一对外国夫妇手持地图，神情茫然地看向四周。我听见他们用英语嘀咕："Where is the Bell Tower?"

我深吸一口气，走上去："Excuse me, can I help you? The Bell Tower is about 800 meters from here. I can show you the direction."

他们先是一愣，随即露出惊喜的笑容。我把路线指给他们，又用手机翻出地图标注了几处景点，连公交路线也一并说清楚了。临走，那位女士握了握我的手，说了一声："Thank you, you're so kind. China is wonderful."

那六个字，像一道光，打进我心里。《礼记》有云："礼者，天下之纲纪也。"礼，不只是中国的传统，也是我们走向世界时最好的语言。那一刻，我忽然明白，自信、从容、热情的我们，就是中国最好的样子。

这是第三枚证据，也是最重的一枚。

---

三枚贝壳，拾进口袋，抵达心里。

15岁，我没有惊天动地的故事，只有这些平凡的瞬间：一锄泥土、一袋菜蔬、一次指路。但我相信，美，就藏在这些细小的行动之间，藏在每一次弯腰、每一声回应、每一个鼓起勇气开口的瞬间。

这是我初中最漂亮的书签。翻过去，是高中崭新的扉页，和更广阔的美正在等待被收集。

---

## 五、范文评析

**总体评价**：结构精巧，三层递进有层次感，细节真实动人，语言干净有质感，约900字，整体达中考**52～55分优秀层次**。

**亮点解析：**

| 维度 | 体现 |
|------|------|
| **立意精准** | "美"定位为品格美，三个维度（孝悌→善良→担当）层层递进，由家庭到社会到国家，境界不断打开 |
| **"证据"意识强** | 每一层均有具体场景、具体细节、具体感受，真正做到了"收集证据"而非泛泛抒情 |
| **结构匠心** | "三枚贝壳"的比喻贯穿全文，与题目"书签""扉页"的意象形成呼应，结构浑然一体 |
| **细节有温度** | "父亲的手掌宽厚而粗糙，像树皮，也像盾"——以比喻写人，形象而有情感张力 |
| **名句嵌入自然** | 引用孟子、《礼记》均在情节推进到感悟处，顺势引出，不显堆砌 |
| **结尾与题目呼应** | 末段"书签""扉页"直接回扣题目中的意象，照应完整，收束有力 |

**可提升之处：**
- 第二个场景（帮老奶奶）可再增加一个对话细节，使人物形象更丰满；
- 三个场景的篇幅略显均等，可适当详写第一个或第三个，形成详略对比，节奏更有变化。

**核心写作技法总结：**
> 此类"收集证据"式命题的核心：**不要写"我很美"，要让读者自己得出"你很美"的结论。** 选取2～3个真实可信的生活片段，用具体细节说话，让行动替代表白，这就是初中作文最高级的表达方式。
