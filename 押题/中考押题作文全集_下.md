# 押题12：墙的另一边

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题目 | 《墙的另一边》（命题记叙文） |
| 核心关键词 | "墙"——可实可虚，实指物理的墙，虚指心理隔阂、误解、偏见；"另一边"——未知的、被遮蔽的、对立的 |
| 命题陷阱 | ①只写邻居之间的隔阂，缺乏深度；②"墙"的象征意义挖掘不足 |
| 审题要点 | "墙"须有象征意义，"另一边"须有一个出乎意料的发现，完成一次认知的突破 |

---

## 二、写作思路

以"我家院墙旁的邻居老人"为切入点。"我"长期因为各种声音和印象，对墙那边的老人有偏见，某一天因偶然机会翻过或绕过"墙"，发现了一个与想象截然不同的世界，完成对偏见的打破。

---

## 三、写作大纲

| 段落 | 设计内容 |
|------|----------|
| 开头 | 描述那堵灰砖墙，引出"另一边"的神秘感 |
| 第一段 | 交代对墙那边老人的印象：沉默、不近人情，经常皱眉 |
| 第二段 | 某天皮球飞过墙，不得不去敲门讨要 |
| 第三段 | 门打开后，看见老人在给一株植物浇水，院中意外的美 |
| 第四段 | 老人将球还我，顺带聊了两句，打破了"我"的所有预设 |
| 结尾 | 揭示：墙，有时候是我们自己砌的 |

---

## 四、范文

### 墙的另一边

那堵墙是灰色的，有些地方的砖缝里生着细细的青苔，像岁月留下的痕迹。我在这条弄堂里住了十三年，却从来没有认真想过墙的另一边是什么。

只知道那里住着一个老头儿。

他不怎么出门，偶尔在弄堂里碰见，也是低着头走路，不打招呼，脸上的表情像块石头，永远沉甸甸的。邻居们私下说，那老头儿性格怪，脾气硬，也没有什么亲人来往。我从小就对他抱有一种莫名的戒备，走路经过他家门口，总是不自觉地加快脚步。

直到那天，我的皮球越过了那堵墙。

放学后在弄堂踢球，一脚踢歪，皮球飞上了墙头，咚的一声落在了墙的另一边。我站在墙下，仰头看了看，那堵墙比我高，跳也跳不上去。左想右想，只能硬着头皮去敲那扇生锈的铁门。

门开了。老人站在门口，眼神打量了我一下，没有说话。

我张了张嘴，正要说球的事，目光却不由自主地飘过他的肩头，飘进了那方小院。

我愣住了。

那院子不大，但种满了花——有叫得出名字的月季、栀子，也有叫不出名字的、缀着细碎紫色小花的藤蔓，顺着墙角的竹架爬满了整面墙。院子正中，一株老梅树虬枝横斜，枝上还挂着几朵枯萎的花苞，但整棵树透着一种苍劲的精神。

老人顺着我的视线看了看，神情里有一丝不易察觉的东西，像是骄傲，又像是温柔。

"皮球，"我回过神，"我的皮球掉进来了。"

他走进去，弯腰从月季花丛旁边捡起了皮球，递给我。"以后踢小一点，"他说，声音很低，却不是凶的，是那种叮嘱的语气，"花盆容易碎。"

我接过球，点了点头，"谢谢爷爷。"

他嗯了一声，关上了门。

走回去的路上，我回头看了看那堵灰色的墙。墙还是那堵墙，可我忽然觉得，它矮了一些。

那一刻我才明白，有时候拦住我们的那堵墙，不是砖砌的，而是我们自己叠起来的偏见和猜测。我们站在墙的这一边，对另一边充满戒备，却从未想过推开那扇门去看一看。

苏轼曾说："横看成岭侧成峰，远近高低各不同。"换一个角度，换一扇门，你所看见的世界，可能和你以为的截然不同。

那之后，我再经过他家门口，不再加快脚步了。有时候，我甚至停一停，隔着墙，闻一闻从院子里飘出来的栀子花香。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **"墙"的虚实结合** | 物理的墙对应心理的偏见，象征意义自然融入叙事 |
| **院子场景惊艳** | 月季、栀子、老梅，用细节打破读者的预期，与前文"沉默老人"形成反差 |
| **老人刻画克制** | 没有煽情，几句简单的话，人物反而更立体真实 |
| **引用苏轼诗句** | "横看成岭侧成峰"用于揭示主旨——换角度看世界，贴切自然 |
| **结尾余韵悠长** | 最后"闻栀子花香"一句，以细节结尾，意蕴深长 |

**得分预测**：50~53分

# 押题13：藏在民俗里的欢乐

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题目 | 《藏在民俗里的欢乐》（命题作文，讲给伙伴听） |
| 写作对象 | 伙伴（同龄人）——语言应活泼、有趣、有画面感 |
| 写作目的 | 让伙伴"体会民俗里的欢乐与童真" |
| 命题陷阱 | ①选的民俗场景模糊，不具体；②语言太正式，忘了"讲给伙伴听"的语气 |
| 审题要点 | 须选一个具体民俗场景，语言要生动、口语化而不失文采，重在"趣"字 |

**本文选择**：元宵节舞龙灯

---

## 四、范文

### 藏在民俗里的欢乐

嗨，我要跟你讲一件让我至今想起来还忍不住笑的事——跟着龙灯跑的那一年元宵节。

那年我大概八九岁，正是觉得什么都好玩、什么都要往上凑的年纪。元宵节前两天，村里就开始准备舞龙了：长长的龙身是用竹篾扎的骨架，外面蒙着金黄和朱红的彩纸，里面每隔一段插一根蜡烛，一点上，整条龙就从里到外透着一种暖洋洋的金色，在夜里看，真的像一条活的龙。

出龙那天是元宵节正晚。锣鼓一响，我就从饭桌上窜下来，饭都没吃完，跟着邻居家的孩子一起跑了出去。

龙队浩浩荡荡地从祠堂门口出发，前面是两个人举着龙珠引路，后面十几个壮汉举着龙身，步子迈得整齐，龙就在夜色里起伏蜿蜒，像真的在游动。锣鼓声震天，鞭炮的硝烟弥漫在空气里，我们这群孩子就在队伍旁边跟着跑，看见龙盘绕的时候一起哇的叫出声，看见火星溅出来了赶紧往后跳，看见领头的大伯向我们招手，有胆大的孩子甚至跑上去摸了一把龙鳞——是厚纸糊的，摸起来粗粗的，却觉得神圣极了。

最狼狈的事发生在我身上。我一路跑一路看，完全忘了看脚下，在一个转弯处一脚踩进了路边的水坑，扑通一声，鞋子和裤腿全湿透了。旁边的小伙伴哄堂大笑，我也顾不上难堪，抖了抖腿，照样继续跟跑——那时候哪里管什么脏不脏，龙还在前面游呢！

跑了整整一条街，汗出透了，鞋子湿透了，耳朵被鞭炮震得嗡嗡响，回家时被妈妈追着数落，说衣服洗不干净了。可我当时心里是满的，满得要溢出来——那种感觉，现在想描述，只能说：像烟花在胸口炸开了。

"爆竹声中一岁除，春风送暖入屠苏。"王安石写的是春节，但我总觉得那句话也属于元宵，属于那一夜锣鼓喧天、龙灯流光的时刻。

你有没有这样的元宵？有没有跟着什么热闹的队伍一路傻跑过？我希望我这个故事，能让你想起那段藏在民俗里的、湿了鞋也要继续跑的欢乐。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **语气贴合写作对象** | 开篇"嗨，我要跟你讲"完全是朋友间说话的口吻，符合"讲给伙伴听"的要求 |
| **民俗场景具体细腻** | 龙灯的制作工艺、出龙过程、孩子们的反应，细节丰富 |
| **趣事选材有亮点** | 踩水坑一幕幽默真实，增加文章的童趣和亲切感 |
| **情感真挚** | "像烟花在胸口炸开了"一句，将童年的快乐表达得生动传神 |
| **引用王安石诗句** | 由春节意象引申到元宵节，拓展了引用场景，不生搬硬套 |

**得分预测**：49~52分

# 押题14：在困境中，我选择____

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题型 | 半命题记叙文，向校报投稿 |
| 补全建议 | 《在困境中，我选择坚持》《在困境中，我选择放下》《在困境中，我选择开口》 |
| 命题陷阱 | ①"困境"描写不充分，缺乏挣扎感；②"选择"发生得太顺利，无过程 |
| 审题要点 | 重在写"选择"的过程——纠结、抗拒、转变，最终选择某一行动，并通过该行动走出困境 |

**本文选择**：《在困境中，我选择开口》

---

## 四、范文

### 在困境中，我选择开口

那段时间，我处在一种很奇怪的状态里——嘴巴是关着的，心里的话却越堆越多。

初二那年，因为成绩下滑，我被调换了座位，离原来要好的同学越来越远。与此同时，课业压力一天比一天大，好几门功课的进度我都跟不上，数学课上老师讲的内容仿佛用的是外星语言，我坐在那里，抄下公式却不知所以然。

我没有告诉任何人。不是没机会，而是开不了口。我总觉得，开口说"我不懂"是一种示弱，是把自己的软肋亮给别人看，万一被人笑话怎么办？于是我把不懂埋起来，假装自己明白，一道题一道题地硬背答案，却越来越混乱。

有一天，晚自习快结束时，我在演草纸上算了第七遍同一道题，还是错的。我把笔一摔，盯着那张写满了划掉又重写的演草纸，第一次认真问了自己一个问题：这样下去，能到哪里？

那天回家路上，我走得很慢。月亮挂在楼栋之间，冷白色的光打在地上，我想起老师之前说的话——"不懂就问，这是学生最大的权利。"那句话我听过很多次，但那一晚，它第一次真的打进了我心里。

第二天上午，我鼓起勇气，在课间走到了数学老师的办公室门口，站了两分钟，然后推门进去，说："老师，我有几道题不太懂，可以讲一下吗？"

就这一句话，我为它积攒了几乎一个月的勇气。

老师没有露出任何意外或失望的表情，只是说："来，坐下。"

那个课间，他把我卡住的三个知识点从头讲了一遍，画了图，举了例子。我坐在那里，感觉脑子里什么东西松动了，一块块积压的石头慢慢挪开位置，光透了进来。

开口，只是一瞬间的事。但我在那个瞬间之前，独自挣扎了太久。

孔子说："知之为知之，不知为不知，是知也。"这句话我早就背过，却在那一次开口之后，才真正明白了它的重量。困境有时不是因为能力不够，而是因为你把一扇本可以推开的门，死死关着，不让人进，也不让自己出。

那一次之后，我学会了开口。向老师，向同学，向任何可以帮助我的人。成绩没有一夜之间飞升，但那种被困住的感觉，慢慢散了。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **困境描写充分** | 数学跟不上、不敢开口的心理、第七遍算错题，困境层次清晰 |
| **"选择"过程有挣扎** | 在老师办公室门口站了两分钟这个细节，把内心挣扎外化为行动，非常生动 |
| **老师形象得当** | "来，坐下"三个字，简洁而温暖，避免了说教 |
| **引用孔子语录** | "知之为知之，不知为不知，是知也"贴合主旨，引用有力 |
| **主旨不落俗套** | 不是"努力就能成功"，而是"开口是走出困境的第一步"，更真实深刻 |

**得分预测**：48~51分

# 押题15：写给2036年的故乡明信片

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题型 | 文体不限，有想象空间，可写信、散文、记叙文 |
| 核心关键词 | "2036年"——未来十年，科技、社会变化；"故乡"——情感根基；"明信片"——简洁、有画面感、有温情 |
| 命题陷阱 | ①空谈科技发展，无个人情感；②忘了"明信片"的体裁特点，写成了长篇叙事文 |
| 审题要点 | 以"明信片"的体裁写，兼顾想象与情感；对未来故乡有具体的展望，同时寄托深厚的情感 |

---

## 四、范文（书信体+散文体混合）

### 写给2036年的故乡明信片

寄往：2036年的你，我的故乡

亲爱的故乡：

我在2026年的春天写下这张明信片，那时候你还是老样子——老街的青石板还有些苔痕，村口的那棵古榕树依然粗壮得三个孩子合抱不过来，傍晚的炊烟从一户户人家的屋顶升起来，带着柴火和饭香的气息。

我想象着，等这张明信片"抵达"的时候，是2036年了。

也许那时候，老街已经修缮一新，青石板还在，但路灯换成了更柔和的光；也许古榕树下的茶摊还在，只是点单方式从老板娘报价变成了扫码自助；也许当年做豆腐花的老师傅已经把手艺传给了他的孙子，那碗豆腐花还是原来的配方，嫩滑，微甜，一口下去，是家乡独有的温度。

我不知道2036年的你，是变得更热闹了，还是更安静了。我只知道，不管你变成什么样子，那些扎根在你土地里的东西不会消失——那些祠堂门口的旧石狮，那条穿过稻田的小渠，那片在黄昏时把天空染成金红色的稻穗，还有那些世世代代在这里生息的人，他们的笑声和方言。

你知道吗，故乡这两个字，是我这辈子写起来最重的两个字。不管我走到哪里，你都在我心里某个不动的地方坐着，像那棵古榕树，像那碗豆腐花，是我无论走多远都可以回头对标的原点。

"举头望明月，低头思故乡。"这句话我从小就会背，长大了才慢慢懂得。思念不是某一个瞬间的涌动，而是某种长久的、低低的、始终在的重量。

2036年的故乡，我希望你好。希望那棵古榕树还在，希望做豆腐花的人还在，希望我回来的时候，你还认识我。

爱你的
一个在远方想你的孩子
2026年春

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **体裁创新** | 书信+散文的混合体裁，明信片风格明显，格式有创意 |
| **时间跨越感强** | 2026年的现状vs2036年的想象，时间纵深感丰富 |
| **想象具体可感** | 点单方式变了但古榕树还在，想象细节真实不空洞 |
| **情感真挚** | "故乡是我无论走多远都可以回头对标的原点"，情感表达有深度 |
| **引用李白诗句** | "举头望明月，低头思故乡"——思乡主题的经典引用，恰到好处 |

**得分预测**：49~52分

# 押题16：我的追光之旅

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题目 | 《我的追光之旅》（命题作文） |
| 核心关键词 | "追光"——追逐某种精神力量或信念；"之旅"——是一段过程，有起点、经历、感悟 |
| 命题陷阱 | ①"光"概念太虚，没有落实到具体人物或事件；②只写追到了，没有写追的过程中的挫折 |
| 审题要点 | "光"须具体化：可以是一个人、一段话、一首诗、一种信念；须有"追"的过程，包括黑暗与曲折 |

---

## 四、范文

### 我的追光之旅

那束光，最初出现在一个秋天的黄昏。

我在图书馆的角落翻到一本旧书，书脊已经掉色，书名是《钢铁是怎样炼成的》。随手翻开，目光落在那段被无数人标注过的文字上——保尔·柯察金关于生命意义的那段话：人的一生应当这样度过，当他回首往事时，不因虚度年华而悔恨，也不因碌碌无为而羞愧。

我在那一段文字前坐了很久，窗外的黄昏慢慢暗下去，图书馆的灯一盏一盏亮起来。那些字里有一种东西，像一束光，照进了那时候的我——一个刚刚经历了一连串失败、觉得自己一无是处的少年。

那段时间我真的很灰暗。数学竞赛初选落榜，写了半年的小说被同学评价为"没意思"，就连早起跑步的计划也在第三天宣告失败。我习惯了把自己的梦想搁置，习惯了在出发之前就告诉自己"算了"。

可那段话戳破了我的逃避。保尔在战场上、在病床上，在失明和瘫痪中，都没有停下来；而我，只是被几次普通的失败打了个趔趄，就想缩进壳里。

那天从图书馆出来，我把那段话抄在了本子扉页上。

追光，从那天开始。

我重新拾起了写作——不是为了给别人看，而是因为我真的想把某些东西留下来。我开始认真备战每一次数学测验，不再只盯着最终的成绩，而是把错误当成地图，一点点找清楚自己究竟卡在哪里。我也重新开始早起，不再给自己太宏大的目标，只是每天多跑一百米。

中间当然还有滑落。有几次测验成绩出来，还是很差，我坐在成绩单前，保尔的那段话会自动在脑海里浮现——不因碌碌无为而羞愧。我还没到"无悔"的时候，但我至少，可以回头看着自己说：我没有停。

追光不是一件可以画出终点的事。或许那束光永远在前面，永远走不到，但走向光的那段路，才是最真实的人生。

正如汪国真所写："既然选择了远方，便只顾风雨兼程。"这是我在追光途中最常念起的话。不是因为前方已经晴朗，而是因为我决定，无论如何，不回头。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **"光"具体化** | 以保尔·柯察金的名言为"光"的来源，有文学依托，立意有层次 |
| **挫折描写真实充分** | 竞赛落榜、小说被批、跑步失败，三件事组合出可信的困境感 |
| **追光过程有起伏** | 不是一帆风顺，有反复滑落，更真实 |
| **引用保尔名言** | 在文中自然引出，作为"光"的来源，不是单纯的装饰性引用 |
| **引用汪国真诗句** | "只顾风雨兼程"收尾，现代诗句语言简练有力 |

# 押题17：超越

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题型 | 话题作文，文体不限 |
| 核心关键词 | "超越"——超越自我（内核），而非超越他人（表面） |
| 命题陷阱 | ①写比赛赢了对手，只有"超越他人"没有"超越自我"；②立意停留在"努力才能超越"的层面，太浅 |
| 审题要点 | 材料已明确指出"超越从前的自我才是真正优秀的人"，立意须聚焦"自我超越"，有具体过程 |

---

## 四、范文

### 超越那个曾经的我

那一次，我超越的不是任何人，只是从前的自己。

初一时，我特别害怕写作文。不是不会写，而是写出来之后，恨不得把本子藏起来，不让任何人看见——因为我自己觉得写得太差，太普通，太像一篇流水账。每次老师当众表扬别人的作文，我就低着头，觉得那个世界和我无关。

那种感觉，叫做自我设限。

初二的时候，一次偶然，我把自己随手写的一段文字发给了网上一个文学社群，没想到有人回复说："这段写得很真实，有感染力。"就这一句话，我反复想了很久。

我开始正视自己的写作。把每次写作文当成一次真正的练习，而不是一次等待被评判的展示；把别人的好词好句当成参考，而不是用来打击自己的标准。一点点地，我发现自己的文字开始有了变化——不再是干巴巴的复述，而是有了温度，有了自己的角度。

期末的时候，语文老师把我的一篇作文当范文读了出来。我坐在位子上，没有低头，而是认真听着老师读出那些我写下的句子，心里有什么东西，咔哒一声，轻轻断裂了——是那道把我关起来的锁，开了。

那道锁，从来不是别人给我加上的，是我自己挂上去的。

"自胜者强。"老子这句话，我是先背下来的，后来才真正懂得。真正强大的人，赢得不是排行榜，而是昨天的那个自己。每一次你做到了曾经认为自己做不到的事，你就完成了一次超越。

超越，从来不需要擂台和掌声，它发生在某个平凡的清晨，当你睁开眼睛，决定今天比昨天多往前走一步的那一刻——那就是超越的开始。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **立意精准** | 紧扣"超越自我而非他人"的主旨，不偏题 |
| **"自我设限"概念有深度** | 把恐惧写作的根源归结为"自我设限"，立意比"我努力了就成功了"更有思想性 |
| **转折自然可信** | 一句陌生人的评价触发改变，比大道理更真实 |
| **引用老子"自胜者强"** | 道家经典引用精准，强有力地呼应"超越自我"的主旨 |
| **结尾余味深长** | "多往前走一步的那一刻"作为超越的定义，平实而深刻 |

**得分预测**：48~51分

# 押题18：执着

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题目 | 《执着》（命题作文，讲述成长故事） |
| 核心关键词 | "执着"——坚持不舍、不轻言放弃，但须有故事支撑，不能泛泛而谈 |
| 命题陷阱 | ①写名人案例，无个人成长故事；②"执着"对象太大，如"执着于梦想"，无具体细节 |
| 审题要点 | 须写自己的成长故事；"执着"的对象要小而具体，如练一首曲子、坚持一项运动、钻研一道难题 |

---

## 四、范文

### 执着

那架钢琴买回来的时候，我六岁。

妈妈说学钢琴好，可以锻炼手指协调性和乐感。我那时候不懂这些，只觉得黑白键盘很漂亮，兴冲冲地答应了。

但真正开始练，才知道有多难。

光是手型就练了整整两个星期——手指要拱起来，像个小山丘，不能塌，不能僵。我一次次被老师纠正，一次次回家对着镜子练，练到手腕酸了，练到妈妈叫吃饭了，还不肯停。

那时候的我，对钢琴有一种执着，简单而纯粹。

可到了小学五年级，执着开始动摇了。那年要考钢琴七级，曲目是一首舒伯特的奏鸣曲，有一段左右手分工复杂、速度又快的段落，我练了三个月，还是频繁弹错。每次练到那一段，手指就像不受控制，该停的地方停不住，该快的地方慢下来。我开始烦躁，开始逃避练琴，开始在心里问自己：这条路，我还要走多久？

有一天下午，我又卡在那一段了，把谱子一摔，站起来走开。爸爸坐在旁边，没有说话，只是走过去，把谱子捡了起来，重新放回谱架上，然后说了一句："再试一遍。"

就这三个字。

我不知道为什么，看着那谱架上的乐谱，忽然觉得眼眶有点热。我走回去，深呼一口气，坐下来，把那一段慢慢再弹了一遍。

考级那天，那段最难的部分，我弹下来了。

七级证书我放在书桌的抽屉里，有时候翻出来看一眼，想的不是证书本身，而是那个下午，摔了谱子又捡回来，深呼一口气再弹的那个自己。

"锲而舍之，朽木不折；锲而不舍，金石可镂。"荀子的话，我是后来才读到的，但那个感受，那个"再试一遍"的感受，我早就亲历过了。

执着，不是没有想过放弃，而是想过放弃，又重新坐了回去。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **执着对象具体可感** | 以练钢琴的具体困难为线索，"执着"有了实在的重量 |
| **爸爸的话简练有力** | "再试一遍"三字，比任何大道理都有力，人物形象简洁鲜明 |
| **执着的定义有深度** | "想过放弃，又重新坐了回去"——这才是真实的执着，不是英雄主义 |
| **荀子引用经典贴切** | "锲而不舍，金石可镂"作为执着最有力的注脚，引用得当 |

**得分预测**：49~52分

# 押题19：重读的力量——从《钢铁是怎样炼成的》的再阅读说起

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题型 | 命题作文，从三本书中选一本写"重读"体验 |
| 本文选择 | 《钢铁是怎样炼成的》 |
| 核心关键词 | "重读"——二次阅读，与第一次读的对比，新的发现与感悟；"力量"——重读带来的精神收获 |
| 审题要点 | 须写出两次阅读的不同感受，揭示"重读"为何有力量：随成长阅历的加深，同一本书给出了不同的回答 |

---

## 四、范文

### 重读的力量——从《钢铁是怎样炼成的》的再阅读说起

第一次读《钢铁是怎样炼成的》，是小学五年级。

那时候我只记住了保尔，记住了他的那段关于生命意义的宣言，把它背下来，写进了作文里。那本书在我那时的理解里，是一个英雄的故事——他打仗，他受伤，他失明，他还是活下来了，还是在写作。我觉得那很厉害，很遥远，是一种我无法企及的高度。

第二次读，是初三的寒假。

我不知道是什么让我重新翻开那本书。那段时间压力很大，模拟考接二连三，成绩起伏让我焦虑。有一天晚上睡不着，随手从书架上抽出来，就翻开了。

这一次，我读出了完全不同的东西。

我不再只看保尔的"伟大"，而开始注意到他的脆弱——他也曾绝望，也曾在病床上想过轻易放弃，也曾被人误解、被命运捉弄，在无数次几乎要被压垮的时刻，选择了站起来。那些挣扎的段落，第一次读时我快速翻过去，第二次读时，我字字句句地读，读到一种真实的颤抖。

那段最著名的话，这次读来，也不再只是一句格言：

"人的一生，应当这样度过……"

这次，我把那段话与自己的处境对标——我有没有虚度？我有没有在本可以做什么的时候，选择了什么都不做？那个问题刺穿了我，让我在书页前坐了很久。

重读的力量，也许就在于此。第一次读，你读的是故事；第二次读，你读的是自己。那本书没有变，但你变了，于是同样的文字给了你全新的回答，有时候比第一次读更深，更重，更疼。

正如朱熹所说："读书百遍，其义自见。"不只是百遍，是每一次带着不同生命经历的重读，都是与作者和自己的一次新的相遇。

那本书现在还放在我的书桌上。我想，等我再长大一些，我还会再读一遍。那时候它又会告诉我什么，我现在还不知道。但我知道，它一定还有什么等着我去发现。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **两次阅读对比清晰** | 小学vs初三，感受从"英雄故事"到"读自己"，对比有力 |
| **重读意义揭示深刻** | "第一次读故事，第二次读自己"，是文章最精彩的观点 |
| **引用原著名句自然** | 保尔名言在文中二次出现，第二次赋予了新的理解层次 |
| **引用朱熹"读书百遍"** | 以此支撑"重读有力"的主旨，引用恰当 |
| **结尾留有余地** | "等我再长大些，还会再读一遍"，意蕴深长，给人以启发 |

**得分预测**：49~52分

# 押题20：牵动我心的____

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题型 | 半命题记叙文 |
| 补全建议 | 《牵动我心的那声叫卖》《牵动我心的那片夕阳》《牵动我心的那双手》 |
| 命题陷阱 | ①补全太平，如"牵动我心的歌声"（与押题06重复）；②有触动没有叙事，或有叙事没有触动 |
| 审题要点 | 须写出"触动"的瞬间，并通过后续叙写揭示为何牵动——须有情感的纵深 |

**本文选择**：《牵动我心的那双手》

---

## 四、范文

### 牵动我心的那双手

我总是不经意就想起外公的那双手。

那是一双做了几十年泥瓦匠的手——粗糙，宽厚，指节因长年握泥刀而变得有些变形，手背上纵横着深深浅浅的纹路，像一幅被反复折叠的地图。

我第一次认真看那双手，是他帮我补院墙的那个夏天。

我家院墙有一处长年漏水，墙皮鼓起来又脱落，坑坑洼洼的。外公来住的那两天，他不声不响地搅了一桶水泥，蹲下身，用泥刀把那些坑洼一处处刮平。我蹲在旁边看，第一次近距离地注意到他那双手：泥刀在手里好像是活的，轻重缓急，全凭那双手的感知。抹平一道，回手轻刮，再一道，再刮，纹理均匀、服帖，像是有什么无声的语言在手与墙之间流动。

"外公，你做这个几十年了？"我问。

"嗯，"他说，眼睛不离手边，"从你外婆还没过门就开始了。"

那句话让我愣了一下。那双手，已经工作了超过四十年。四十年间，它们砌过多少墙，修过多少屋，磨破了多少层皮，积下了多少的老茧，我无从想象。

那天晚上，外公吃饭时把那双手放在桌上，我偷偷地看了很久。那双手和他的脸一样，写满了某种东西，不是沧桑，而是——踏实。是那种什么都见过、什么都干过、没有什么可以让他慌乱的踏实。

后来外公身体不好，不再爬架子了，但他还是会在院子里修修补补，哪里松了、哪里裂了，都逃不过他的眼睛。那双手，停不下来。

"谁知盘中餐，粒粒皆辛苦。"这首诗我背了很多年，但那天看着外公那双手，我才真正明白，辛苦不是一种叙述，而是一种刻痕——刻在岁月里，也刻在每一个普通劳动者的手上。

那双手牵动了我的心。它让我知道，平凡的劳动里，藏着一种无声的尊严，值得我们永远看见，永远铭记。

---

| 亮点 | 说明 |
|------|------|
| **"手"这一意象选材独特** | 以手作为情感载体，区别于常见的"眼神""笑容"，视角新颖 |
| **劳动场景描写精细** | 泥刀的使用方式、"轻重缓急全凭手的感知"，动作描写专业可信 |
| **外公的话有分量** | "从你外婆还没过门就开始了"，一句话道出四十年岁月，克制有力 |
| **"踏实"这个词的使用** | 不用"沧桑"而用"踏实"，显示出作者对人物的独到观察 |
| **引用悯农诗句贴切** | "粒粒皆辛苦"用于升华劳动尊严的主旨，贴切而有说服力 |

# 押题21：微光亦温暖（微光亦暖心）

## 一、考题分析

| 要素 | 内容 |
|------|------|
| 题目 | 《微光亦暖心》（命题记叙文）——题目与标题有细微差异，以题目要求为准 |
| 核心关键词 | "微光"——细小的光、不显眼的善意；"亦"——强调虽小却真实；"暖心"——情感上的温暖 |
| 命题陷阱 | ①写的不是"微小"的事，而是宏大的感动；②"暖"字体现不出来，只叙事不写感受 |
| 审题要点 | "微"是关键——须写细小、不起眼的善意或美好，却带来真实的温暖感 |

---

## 四、范文

### 微光亦暖心

那天下大雨，我没带伞。

放学铃一响，走廊上的同学们纷纷拿出伞，三三两两地往外走。我站在门口，看着院子里密密的雨帘，书包抱在胸前，有些茫然。

正犹豫着要不要冒雨跑的时候，有人从旁边走来，在我身边停住了，递给我一把伞——是一把普通的折叠伞，黑色的，有一点旧了，伞尖处有小小的磨损。

"给你，我有另一把。"

是隔壁班的一个女生，我叫不出她的名字，只是偶尔在操场上见过几次。她把伞塞给我，没等我说谢谢，就转身走了，从书包里拿出另一把透明的伞，打开，消失在雨里。

就那么一秒钟的事。

我站在走廊里，手里握着那把黑色的旧伞，心里忽然有一种说不清楚的感觉——不是感动，或者说，不只是感动，而是一种更轻柔的东西，像灰蒙蒙的下雨天里，忽然透进来一缕光，不刺眼，不张扬，就只是亮了那么一点。

我撑开伞走进雨里，脚步比来时轻了不少。那把伞我用了一路，收起来的时候，有点舍不得，像是把那个陌生女生的好意也跟着收起来了。

那天晚上，我把这件事写进了日记里，只写了一句话："有人在你最需要的时候，递来一把伞。"

这样的事，不会上新闻，不会被人记住，甚至那个女生本人，也许第二天就忘了。但对那一刻站在走廊里的我来说，它是真实的温暖，是阴雨天里一道微小的光。

"随风潜入夜，润物细无声。"杜甫写春雨，可这句话让我想到了人世间那些无声的善意——它不求回报，不求被看见，却真真切切地温暖了某一个人的某一个瞬间。

微光，不因为小而轻，正因为小，它才显得真。

世界上需要的，从来不缺轰轰烈烈的大光；真正让人心里暖起来的，往往是那些不经意的微光。

---

## 五、范文评析

| 亮点 | 说明 |
|------|------|
| **"微"字贯穿始终** | 陌生人借伞的小事，没有认识、没有名字，是真正的"微"光 |

## 二、范文

### 真诚，是最贵的门票

2026年五一档，一部成本1400万、全素人出演的小电影《给阿嬷的情书》，凭借观众口碑悄悄逆袭，最终总票房突破2.3亿，豆瓣评分9.1分，成为近十年国产电影口碑天花板。

没有流量明星，没有华丽特效，甚至95%的对白是普通话观众听不太懂的潮汕方言。这样一部电影，凭什么让全国观众哭着走出电影院？

答案只有两个字：真诚。

那部电影讲述的，是一个孙辈向临终的阿嬷道别的故事。故事朴素，情节简单，没有什么戏剧性的冲突和高潮，有的只是家常的琐碎、方言里的温柔，还有那种任何人都懂的——对亲人的爱与不舍。

白居易写诗，"其辞质而径，欲见之者易谕也"，意即语言要质朴直接，让人一读就懂。《给阿嬷的情书》做到了这一点——它不试图讨好，不试图制造眼泪，只是把一段真实的情感，诚实地呈现在你面前，然后，眼泪自己流了出来。

这让我想到，真诚，是一种久违了却从未失效的力量。

我们生活在一个信息喧嚣的时代，什么都讲包装，什么都讲流量，什么都要在三秒内抓住眼球，否则便被划走。在这样的时代里，《给阿嬷的情书》的成功，是一次逆势而为的胜利——它用最慢的节奏，最朴素的语言，最真实的情感，重新赢回了观众的心。

它告诉我们：真正打动人心的，从来不是精心设计的催泪点，而是那种让人感觉"这说的就是我"的真实感。

这也是写作的道理，也是做人的道理。

"感人心者，莫先乎情。"白居易这句话，是对文学的期望，也是对一切表达的期望。技巧可以学，特效可以买，但真诚，是花钱买不到的。

《给阿嬷的情书》的票房，是观众用真心换来的。

而那种真心，从来都是最贵的门票。

| **联系写作与做人** | "这也是写作的道理，也是做人的道理"，将热点拓展到普遍意义 |
| **适用题目** | 可作为押题02（解锁/解锁真诚之门）、押题03（最好的礼物是真诚）等题的素材 |
